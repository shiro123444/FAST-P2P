## 🎯 效果预览

### Hero Section
```
标题: 从左滑入 (slideInLeft)
副标题: 从左滑入 (延迟 100ms)
安全特性: 从下淡入 (延迟 200ms)
右侧卡片: 从右滑入 (slideInRight)
```

### 响应式断点
```
< 768px:  单列，居中对齐
768-1024px: 双列，左对齐
1024-1200px: 双列，增加间距
> 1200px: 双列，最大宽度 1600px
```

### 排版改进
```
Hero 标题: 48px ~ 120px (响应式)
Section 标题: 32px ~ 64px
副标题: 18px ~ 24px
正文: 16px ~ 18px
行高: 1.5-1.6 (优化可读性)
最大宽度: 65ch (舒适阅读)
```
